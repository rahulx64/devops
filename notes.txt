filters in linux
üß© What is a filter in Linux?

In Linux (including CentOS), a filter is simply a command-line program that takes input, processes it, and produces output.
Filters are often used in pipelines (connected by the | operator), where the output of one command becomes the input of another.

-------------------------redirectin-----
üß≠ What is Redirection?

Redirection in Linux means changing where input and output go.

Normally:

Input comes from your keyboard.

Output goes to your screen (terminal).

With redirection, you can:

Send output to a file instead of the screen.

Take input from a file instead of the keyboard.

Combine or separate standard output and error output.
| Symbol | Meaning                 | Example                   | Description          |              |                             |
| ------ | ----------------------- | ------------------------- | -------------------- | ------------ | --------------------------- |
| `>`    | Output (overwrite)      | `echo hi > file.txt`      | Write output to file |              |                             |
| `>>`   | Output (append)         | `echo hi >> file.txt`     | Add output to file   |              |                             |
| `<`    | Input                   | `sort < file.txt`         | Take input from file |              |                             |
| `2>`   | Error output            | `ls /bad/path 2> err.txt` | Write errors to file |              |                             |
| `2>&1` | Combine output + errors | `cmd > out.txt 2>&1`      | Put both in one file |              |                             |
| `      | `                       | Pipe                      | `cat file            | grep "name"` | Send output to next command |

-----------------sudo--------------------
üîê What is sudo in Linux?

sudo means ‚Äúsuperuser do‚Äù.
It allows a regular (non-root) user to run commands with root (administrator) privileges.

You use sudo when you need to:

install software

modify system files

add/remove users

start/stop system services

change system configuration

access protected directories

Without sudo ‚Üí "Permission denied"
With sudo ‚Üí Command runs as root
